# Contributing to Bubble(ဘဘယ်လ်) <img src="https://bubblemusic.vercel.app/bubblelogo.svg" width=100 height=27.747  />

Thank you for your interest in improving this project!

## 📌 Contribution Workflow

- Fork the repository on GitHub
- Clone your forked repository

```bash
git clone https://github.com/your-username/bubblemusic.git
```

```bash
cd bubble
```

- Create a new branch

```bash
git checkout -b feature-name
```

- Make your changes and test locally
- Commit your changes

```bash
git commit -m "Add your message"
```

- Push your changes

```bash
git push origin feature-name
```

- Open a Pull Request

## Code Guidelines

- Follow existing coding style
- Write clean and readable code
- Keep changes focused on one feature
- Test your changes before submitting PR
- try to use existing colors for UI changes if possible

## UI Changes

If you update UI components:

- Add screenshots if possible
- Add video demonstration if necessary
- Explain changes in Pull Request description

## ⭐ Thank You!

Your contribution helps improve Bubble!
